The basics

For every object in the TargetMine database there is a report page which provides all the data that TargetMine has available for that object along with additional information such as the protein domain viewer for protein entries.

All report pages have the same basic structure: The top part provides summary information for every object, whereas the lower part, Further Information by category has more detailed information divided into different data types or categories

As an example the report page for the APOB gene is shown.

Template queries are run for this object when the page is loaded.

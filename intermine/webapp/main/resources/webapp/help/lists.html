Upload

TargetMine can operate on lists of data. Lists can then be analysed on a list analysis page, or can be used in (template) queries. The Upload page allows you to create a list yourself. The list can contain a mix of identifier types (eg symbols, names, organism database id,) but it cannot contain different object types (eg a list of genes and proteins is currently not possible).

Identifiers must be separated by commas, spaces, or lines. If you have an identifier that contains spaces, use double quotes, eg: 
  eve, zen, "Enhancer of zeste", CG32211
not: 
  eve zen Enhancer of zeste CG32211

View

The View page shows all public lists, as well as your own (marked with 'my') - if you are logged in. Clicking on a List name will show the List analysis page for this list.

You can merge lists, subtract lists and find common members between lists.
